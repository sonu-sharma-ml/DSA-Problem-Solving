# DSA with Python

A comprehensive collection of Data Structures and Algorithms implementations in Python.

## 📁 Project Structure

```
DSA-with-python/
├── 📂 Array/                      # Array-based problems and algorithms
│   ├── 01_binary_search.py            # Binary Search implementation
│   ├── 03_search_insert_position.py   # Search Insert Position
│   ├── 04_cell_floor.py               # Ceil and Floor operations
│   ├── 07_binary_search.py            # Additional Binary Search
│   ├── 51_Search_in_Rotated_Sorted_Array.py  # Search in Rotated Sorted Array
│   ├── two_sum.py                     # Two Sum problem
│   └── 📂 Linked/                     # Linked List implementations
│       ├── 01_introduction.py         # Linked List introduction
│       ├── 02_create_list.py         # Creating a linked list
│       ├── 02_traversal.py           # Traversing a linked list
│       ├── 03_insert.py              # Insertion in linked list
│       ├── 03_singly_linkedlist.py   # Singly Linked List
│       ├── 04_insert.py             # Additional insertion operations
│       ├── 05_2nd_method.py         # Alternative method
│       ├── 05_middle_linkedlist.py   # Find middle of linked list
│       ├── 06_2nd_method.py         # Alternative method
│       ├── 06_reverse_linkedlist.py # Reverse linked list
│       └── singly_link.py           # Singly Linked List implementation
├── 06_Find_First_and_Last_Position.py  # Find First and Last Position of Element
└── 52.py                            # Additional problem
```

> **Note:** The actual folder in the project is named `Lniked` (typo). The above shows the corrected naming.

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   ```

2. **Navigate to the project directory**
   ```bash
   cd DSA-with-python
   ```

3. **Run any Python file**
   ```bash
   python Array/01_binary_search.py
   ```
